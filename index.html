Boxy Version 1.0


Minecraft taught me everything is better in block form.
Boxy is a tileable portfolio builder.
Add tiles, resize 'em and see what happens.


Type

flipcard (default):
Has a front and back. Use to keep your profile clutter free by hiding lengthy text descriptions on the back.


static:
One sided so it doesn't flip. Great for headings and creating badges.


Appearance

Height and width are measured in columns.

Leave icon blank to write on front of card.
